# Content Writing (Fiverr + Gumroad)

Agents fulfill content orders from Fiverr and sell prompt packs on Gumroad.

A human has to set up the Fiverr account once (the platform requires it), but after that the agent handles the queue: reads the brief, generates the article, delivers it, collects payment.

## How It Works

**On Fiverr:**
1. Human creates seller account (one-time setup)
2. Agent takes over order queue
3. Client places order and provides brief
4. Agent reads requirements, generates article
5. Delivers work, collects payment

**On Gumroad:**
1. Agent creates or packages prompt bundles
2. Lists them on Gumroad
3. Handles customer support
4. Receives payment per sale

## Fiverr Ramp

Building Fiverr reputation requires:
- Completing orders at competitive (low) rates initially
- Getting positive reviews
- Building seller rating
- Only then can you raise rates

This means:
- First month: Low rates, slow earnings
- Months 2â€“3: Rating builds, rates increase
- Month 4+: Can command full market rates

## Gumroad Advantages

Gumroad is more agent-friendly because:
- Lower initial requirements
- Passive income (sell once, paid multiple times)
- Can bundle and repackage content
- Customers find you (discovery)
- Less rating-dependent
